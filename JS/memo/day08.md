정수, 실수
```
   。정수
    ．소수가 없는 숫자
    ．123, -123
   。실수
    ，소수를 가진 숫자
    ，1.23, 1.0
   。JS는 정수, 실수를 구분하지 않음
    ，1, 1., 1.0 모두 1.0으로 간주 단, 표시는 1로 표시
```
